Add custom Open Graph meta tags to your posts for better social media sharing.

== Description ==

Open Graph Tags allows you to customize how your content appears when shared on social media platforms. You can set custom values for:

* og:title - The title that appears when shared
* og:description - The description that appears when shared
* og:image - The image that appears when shared
* og:url - The URL that will be shared
* og:type - The type of content being shared

Features:

* Easy-to-use sidebar panel in the block editor
* Support for image upload and media library selection
* Fallback to post title, excerpt, and featured image
* Validation for optimal social sharing
* Clean uninstall

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/open-graph-tags` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the sidebar panel in the post editor to set your Open Graph tags

== Frequently Asked Questions ==

= What happens if I don't set custom values? =

The plugin will use fallback values:
* Title: Post title
* Description: Post excerpt
* Image: Featured image
* URL: Post permalink
* Type: "article"

= Does this work with custom post types? =

Currently, the plugin only supports regular posts.
